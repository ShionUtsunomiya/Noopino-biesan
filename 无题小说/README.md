# 无题小说

## 前言

我写下的，可能是一本小说,也可能只一篇散文。为什么呢？因为我没有构思好剧情。但为什么它能成为一篇小说?因为我能感受到它灵魂(及主旨)正在成形。仅管它现在没有血肉（剧情、人物），但仿佛一切又已经在那，我只需要边走边拾取，他就可以长大。我朋友说过:当你确定一件事后，无需计划太多，因为计划赶不上变化；也无需担忧和迷茫，因为思路在过程中会渐渐清晰；也无需学习太多，干中学就行了。所谓出发永远比不出发好。

2025-11-07

## 第一章

我拿起桌上的书，发现上面盖有我的名字。妻子问：为什么你会喜欢在书上盖章。我静静地回答：这个世界上仿佛所有东西都不属于我，比如我们曾经的车是租的，我开的小心翼翼，也不敢改装他、贴车膜，因为我知道有一天要还回去。但是唯有书是完全属于我的，我可以肆无忌惮的使用它：给他盖章，在侧面写下书名，给它写注释，乱涂乱花，甚至撕掉、扔掉我不喜欢的书。这时妻子沉默不语，我看向她，她竟然眼眶泛红。我问她为什么，她说可能来大姨妈变得非常感性，她觉得我很惨。

相反，我这一辈子可谓顺风顺水，至于为什么她老是觉得我惨，可能是我从小比较缺爱，毕竟我的家庭一下子被拔高了，破事就会有很多。我也承认我缺爱，幸运的是我找到了一个爱我的人，这何尝是惨呢？所以我可能在一个惨与不惨的叠加态，惨与不惨，只在一念之间。这时我想起了一段记忆，但是这也可能是一个故事。因为在我的回忆里，故事的主体竟然不是我，而我又出现在这段记忆里，这段话有点绕，听我说来，我老婆似乎也来了兴趣，眼眶又透露出好奇。那是在一个风雪天里，我开着摩托车，天太冷太冷，我把我的皮衣反过来穿了，把有毛绒的一面放在外面。我的孩子在我胸口睡着了（这孩子其实是我）。不太记得为什么要把皮衣外穿，据我一个叔叔说这样更防寒，难道是抓绒外套的原理？扯远了，
